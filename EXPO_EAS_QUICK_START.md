# دليل سريع: بناء APK باستخدام Expo EAS

## ⚡ 3 خطوات فقط!

### الخطوة 1️⃣: تثبيت EAS CLI

```bash
npm install -g eas-cli
```

---

### الخطوة 2️⃣: تسجيل الدخول أو إنشاء حساب

**إذا كان لديك حساب Expo:**
```bash
eas auth:login
```

**إذا لم يكن لديك حساب:**
```bash
eas auth:register
```

ستظهر نافذة متصفح لتسجيل الدخول/التسجيل

---

### الخطوة 3️⃣: بناء APK

```bash
cd /home/ubuntu/invoice-pro-app
eas build --platform android
```

**ثم اختر:**
- Build type: `apk` (وليس `aab`)
- Distribution: `internal`

---

## ⏱️ كم من الوقت يستغرق؟

- **التثبيت:** 2 دقيقة
- **التسجيل:** 5 دقائق
- **البناء:** 15-30 دقيقة

**المجموع: حوالي 45 دقيقة** ✅

---

## 📱 بعد انتهاء البناء:

ستحصل على:
1. **رابط التحميل المباشر** للـ APK
2. **رمز QR** لتحميل سريع
3. **ملف APK** جاهز للتثبيت

---

## 🎯 التثبيت على الهاتف:

### الطريقة 1: من الرابط المباشر
```
انسخ الرابط وافتحه على هاتفك
```

### الطريقة 2: من رمز QR
```
امسح رمز QR بكاميرا الهاتف
```

### الطريقة 3: نقل يدوي
```
1. حمّل APK على الكمبيوتر
2. انقله إلى الهاتف عبر USB
3. اضغط على الملف للتثبيت
```

## 🆘 إذا حدث خطأ:

### خطأ: "Not logged in"
```bash
eas auth:login
```

### خطأ: "Build failed"
```bash
# تحقق من الكود
npm install
pnpm install

# ثم حاول مرة أخرى
eas build --platform android
```

### خطأ: "Timeout"
```bash
# الخوادم مشغولة، حاول لاحقاً
eas build --platform android
```
