# تصميم واجهة تطبيق Invoice Pro

## نظرة عامة
تطبيق موبايل احترافي للفواتير يعمل بكفاءة على الأجهزة الذكية بصيغة عمودية (Portrait)، مع التركيز على سهولة الاستخدام بيد واحدة.

## قائمة الشاشات

### 1. شاشة البداية (Splash/Onboarding)
- عرض شعار التطبيق
- زر "ابدأ الآن" للدخول إلى التطبيق

### 2. شاشة الإعدادات (Settings/Company Setup)
- حقول الإدخال:
  - اسم الشركة (اختياري)
  - العنوان (اختياري)
  - رقم الهاتف (اختياري)
  - تحميل الشعار (اختياري)
- زر "حفظ الإعدادات"
- عرض البيانات المحفوظة سابقاً

### 3. الشاشة الرئيسية (Home/Dashboard)
- قائمة تبويبات (Tab Bar):
  - **الفواتير** (الشاشة الافتراضية)
  - **الإعدادات**
- عرض قائمة الفواتير السابقة (إن وجدت)
- زر عائم "إنشاء فاتورة جديدة" (FAB)

### 4. شاشة إنشاء الفاتورة (Invoice Creation)
- **القسم الأول: بيانات العميل**
  - اسم العميل (اختياري)
  - رقم الهاتف (اختياري)
  - البريد الإلكتروني (اختياري)
  - العنوان (اختياري)

- **القسم الثاني: الأصناف والخصم**
  - جدول ديناميكي:
    - اسم الصنف (اختياري)
    - السعر (اختياري)
    - الكمية (اختياري)
    - نوع الخصم (نسبة % / مبلغ ثابت)
    - قيمة الخصم (اختياري)
  - زر "إضافة صنف"
  - زر "حذف صنف" لكل صف

- **القسم الثالث: الملخص**
  - الإجمالي قبل الخصم
  - إجمالي الخصم
  - الإجمالي النهائي
  - عرض التفقيط بالعربية

- **الأزرار السفلية:**
  - زر "اختر قالب PDF"
  - زر "إرسال عبر WhatsApp"

### 5. شاشة اختيار قالب PDF (Template Selection)
- عرض 3 قوالب مختلفة:
  - **القالب 1**: بسيط وكلاسيكي
  - **القالب 2**: احترافي مع ألوان
  - **القالب 3**: حديث مع تصميم عصري
- معاينة لكل قالب
- زر "اختيار هذا القالب"

## تدفقات المستخدم الرئيسية

### تدفق إنشاء فاتورة:
1. المستخدم ينقر على زر "إنشاء فاتورة جديدة"
2. ينتقل إلى شاشة إدخال بيانات العميل
3. يملأ بيانات الأصناف مع الخصم
4. يختار قالب PDF من 3 خيارات
5. يختار "إرسال عبر WhatsApp"
6. يتم توليد PDF وفتح قائمة المشاركة

### تدفق الإعدادات:
1. المستخدم ينقر على تبويب "الإعدادات"
2. يدخل بيانات الشركة
3. ينقر "حفظ الإعدادات"
4. يتم حفظ البيانات محلياً

## الألوان والعلامة التجارية

| العنصر | اللون | الاستخدام |
|--------|-------|----------|
| اللون الأساسي | `#0a7ea4` (أزرق) | الأزرار والرؤوس |
| الخلفية | `#ffffff` (أبيض) | خلفية الشاشات |
| النص الرئيسي | `#11181C` (رمادي غامق) | النصوص الأساسية |
| النص الثانوي | `#687076` (رمادي) | النصوص الثانوية |
| الخطأ | `#EF4444` (أحمر) | رسائل الخطأ |
| النجاح | `#22C55E` (أخضر) | رسائل النجاح |

## مبادئ التصميم

- **البساطة**: واجهة نظيفة وسهلة الاستخدام
- **الاستجابة**: تصميم يعمل على جميع أحجام الشاشات
- **الوضوح**: نصوص واضحة وأيقونات معبرة
- **الكفاءة**: تقليل عدد الخطوات اللازمة لإنشاء فاتورة
- **الملاحظات**: تغذية راجعة فورية للمستخدم (رسائل النجاح/الخطأ)
