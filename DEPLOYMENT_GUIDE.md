## 📱 كيفية الحصول على APK

### الطريقة 1: من GitHub Actions (تلقائي)

1. **اذهب إلى Repository:**
   - https://github.com/oka-soft/invoice-pro-app

2. **انقر على "Actions" tab**

3. **ابحث عن "Build APK" workflow**

4. **انتظر انتهاء البناء (10-15 دقيقة)**

5. **حمّل APK من Artifacts:**
   - انقر على آخر run
   - انقر على "Artifacts"
   - حمّل `invoice-pro-app`

### الطريقة 2: البناء اليدوي

```bash
cd /home/ubuntu/invoice-pro-app
pnpm install
npx eas build --platform android --local
```

## 🚀 تشغيل التطبيق

### على Android:
1. انقل ملف APK إلى هاتفك
2. انقر عليه للتثبيت
3. اسمح بالتثبيت من مصادر غير معروفة إذا لزم الأمر

### على Expo Go (للاختبار):
```bash
cd /home/ubuntu/invoice-pro-app
pnpm dev
# ثم امسح QR code بهاتفك
```
